PunchClockPanda modernization changes

1. Updated compileSdk and targetSdk to 35 for current Play Store target API policy.
2. Updated Android Gradle Plugin to 8.7.3 and Gradle wrapper to 8.9.
3. Updated Kotlin plugin to 1.9.24.
4. Updated library dependencies to newer stable AndroidX / Material versions.
5. Explicitly set android:exported on all activities for Android 12+ compliance.
6. Kept the app offline-only. No INTERNET permission was added.
7. Kept CSV export and Android share flow via FileProvider.
8. Kept release minification off to reduce behavior-change risk for first release.
9. Bumped versionCode to 2 and versionName to 1.0.1.

Recommended in Android Studio:
- Open the project and let Gradle sync.
- Use a JDK 17 runtime.
- Build an Android App Bundle (.aab) for Play release.
